#### Mega2560 vs Gameboy

|     |Mega 2560|Gameboy|
|:----|:--------:|:----:|
|CPU| 8-bit 16MHz  |   8-bit 4.19MHz   |
|RAM| 8KB SRAM |8KB SRAM    |
|Display| 128X64 OLED  |   160X144 LCD  |

#### 기능
 * 에뮬레이터 기능 제외
